## 使用方法



- 抖音主页 - 分享 - 复制链接 - 发送到电脑

- 点击链接 - 将网址复制后使用浏览器访问

- 打开Devtools - 选择手机预览模式
- 点击"**作品**"，找到加载视频时对应的url
- 运行脚本，依次输入url和用户ID（step2里有）
